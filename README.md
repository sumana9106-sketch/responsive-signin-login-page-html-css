# Login Page UI

A clean, modern, visually appealing and responsive Login Page built using HTML and CSS.

## 📌 Project Overview

This project is a frontend login page designed to provide a clean and modern user interface. It features a glassmorphism-inspired design, responsive layout, hover animations, and interactive form elements.

The project was created to strengthen my understanding of HTML, CSS, Flexbox, responsive design, and UI development.

## ✨ Features

* Modern glassmorphism design
* Responsive layout for different screen sizes
* Login form with email and password fields
* Remember Me option
* Forgot Password link
* Sign Up navigation link
* Button hover animation
* Input focus effects
* Gradient button border effect
* Clean and organized code structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries

## 📱 Responsive Design

The layout adapts to different screen sizes using CSS media queries and flexible sizing techniques.
